# decast-web5

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/VideoWiki/decast-web5.git
    cd decast-web5
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Start the application:

    ```bash
    npm run start
    ```
